# 时速体育

  应用介绍: "时速体育"是一款线上预定越野馆、报名越野车赛事的应用。

  应用功能: 

    【首页】关于越野馆、越野赛事的推荐，以及刺激战场、最热场馆和越野都爱等。
    
    【越野馆】精选成都优质的越野馆，可直接线上预约、线下到店消费使用。
    
    【体育赛事】关于越野的竞技赛事，可直接线上报名，等待审核通过即报名成功。
    
    【预约】预约订单列表，相关订单信息详情，随时可退订订单。
        
    【我的】个人信息、我的收藏、客服热线和相关设置等；
         我的报名(已报名/审核中/审核通过)。
